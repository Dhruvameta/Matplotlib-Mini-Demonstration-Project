# Matplotlib Mini Demonstration Project

A beginner-friendly demonstration project showcasing the fundamental features of the **Matplotlib** library in Python. This project contains a collection of simple programs that illustrate how to create different types of plots and customize them using Matplotlib.

## Overview

The purpose of this project is to understand the basics of data visualization with Matplotlib by implementing commonly used plotting techniques. Each program focuses on a specific concept, making it easy for beginners to learn and experiment.

## Features

- Line Plot
- Bar Chart
- Horizontal Bar Chart
- Scatter Plot
- Histogram
- Pie Chart
- Multiple Plots (Subplots)
- Grid and Axis Customization
- Titles and Labels
- Legends
- Colors, Markers, and Line Styles
- Figure Size Customization
- Saving Figures

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/matplotlib-demo.git
```

2. Navigate to the project directory:

```bash
cd matplotlib-demo
```

3. Install Matplotlib (if not already installed):

```bash
pip install matplotlib
```

## Running the Programs

Run any Python file using:

```bash
python filename.py
```

Example:

```bash
python line_plot.py
```
